## 📌 Projektbeschreibung

In diesem Projekt wurden die Verkaufsdaten eines E-Commerce-Shops aus den Jahren 2019–2024 analysiert. Ziel war es, Trends im Kundenverhalten zu identifizieren und die Wirksamkeit von Rabattaktionen zu bewerten.

**Kernaspekte des Projekts:**
- **KPI-Analyse:** Monitoring von Brutto- und Nettoumsatz sowie Transaktionszahlen.
- **Kundensegmentierung:** Identifizierung der profitabelsten demografischen Gruppen.
- **Promotion-Effektivität:** Analyse des Einflusses von Rabatten auf den durchschnittlichen Bestellwert (AOV).
- **Business Insights:** Erarbeitung konkreter strategischer Empfehlungen für den Shop.
